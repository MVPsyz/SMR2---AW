¿Qué es un index.html?

El archivo index.html es la página principal de un sitio web. Es lo primero que se carga cuando visitas un dominio. A continuación te mostramos su estructura básica:

<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <title>Título de la página</title>
  </head>
  <body>
    <h1>Hola Mundo</h1>
    <p>Este es el contenido principal.</p>
  </body>
</html>

Explicación rápida:

  • <!DOCTYPE html>: Define el tipo de documento (HTML5).
  • <html>: Contenedor principal de todo el contenido HTML.
  • <head>: Contiene metadatos, título, enlaces a estilos, etc.
  • <body>: Contiene el contenido visible de la página.
